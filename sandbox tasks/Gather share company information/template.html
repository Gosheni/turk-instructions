Instructions

This task requires visiting multiple social media and company websites to find relevant information for the securities (i.e company or stock) listed on stock exchanges.
For each security, the name and the ticker along with the exchange will be listed.

Tip: Provided along with each question will be an example search term. Copying this term into a search engine (e.g. Google) should provide useful results.

If the relevant page does not exist, leave the input blank.
For each company the following information should be acquired:

Twitter Page
Format twitter.com/Company
Example twitter.com/coles
Facebook Page
Format facebook.com/Company
Example facebook.com/coles
Website URL
Format https://CompanyURL.com
Example https://colesgroup.com
Investor Relations Page
Format https://CompanyURL.com/InvestorRelationsPage
Example https://colesgroup.com/investors/?page=investors
