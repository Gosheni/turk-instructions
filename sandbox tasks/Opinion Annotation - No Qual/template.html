Fine-Graine​d Opinion Extraction Annotation Task

Instructions

Annotate the following sentence according to Fine-Grained Opinion Extraction guidelines. 

 
Reminders

If you have not yet accepted the HIT, make sure to accept it before you highlight any words.

  • If you have not yet read the annotation guideline document, please do so before attempting any annotation. 
  • Only annotate sentences with an explicitly expressed opinion; do not annotate sentences that have sentiment, but no opinion. For example, 
  •         The customer service was on point!   ← annotate sentences like this
  •         ​The shrimp wasn't a great experience ← annotate sentences like this
  •         If I were you, I wouldn't eat the shrimp. ← don't annotate sentences like this.
  •         I’ll never be going to your store again! ← don’t annotate sentences like this.
  • If a sentence should not be annotated, simply submit the task with no words highlighted. 
  • When in doubt, feel free to ask John about expected behavior on Slack.
