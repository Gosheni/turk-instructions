Instructions

Objective: Determine the topics in each Reddit comment and the emotions that are expressed toward them.

Given a comment extracted from a Reddit thread, identify the topics that are present in the text. You will be presented with at most two automatically generated options for topics. These are meant to facilitate the annotation process and you can select by clicking (the text should appear in one of the text boxes below, which can be modified as well). After listing them, select or provide an emotion that is expressed towards each topic in the comment.

To select an emotion, you must first select a sentiment. After this is done, the appropriate emotion selector will appear.

If the desired emotion isn't provided in the selector, please write a response in the text box.

Note that masked tokens may appear in comments. Proper names are masked with [NAME] and religious terms are masked with [RELIGION]. If a topic involves masked tokens, please do not hesitate to use them in your responses.
