Website data collection - Instructions (Click to collapse)

Your task is to collect scholarship data for unique scholarships. There are eight pieces of data that need to be identified and recorded. Each website URL is unique and has data in different areas of the page. You will skim the page looking for relavent data.

Tips
  • Estimated time to index one scholarship is 1 minute.
  • Copy and paste the scholarship title directly from the website
  • If the scholarship does not say anything about a required specific Ethnicity, check all the boxes.
  • If the scholarship does not say anything about a required specific current School Year, check all the boxes.
  • If the scholarship does not say anything about a US Citizenship, select "No".
  • If the scholarship is being awarded to Multiple applicants, select "Yes". (Ex: Three awards of $1,000 each.)
    • The first question is very important!

    On occasion you will come across a web page that has multiple unique scholarships listed. In this case, make the correct selection form the first question and submit the HIT. There is no need to gather more data.
