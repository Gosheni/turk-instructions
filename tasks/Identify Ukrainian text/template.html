Say whether Tweets are written in Ukrainian or not

  • This is a task for fluent speakers of Ukrainian.
  • Below are some short texts along with some decision options.
  • Please read each short text on the left.
  • On the right, mark each text for whether it is written for speakers of the Ukrainian language.
  • Also mark it as Ukrainian if:
    • it is a dialect of Ukrainian
    • it is in Ukrainian but written with English letters [abcdef...]
    • it is written for Ukrainian speakers but has some other languages mixed in.
  • If in doubt, mark it as Ukrainian if:
    • most of the text is in Ukrainian (excluding URLs, hash-tags, etc.).
    • Otherwise, mark as `Not Ukrainian'.
    • If still not sure, mark as 'Not sure' (rare cases).

Example:

  • "А ще в Києві всі кудись спішать. Всюди. " is Ukrainian
  • "Работна група готви предложенията за реформиране на спешната помощ. " is Not Ukrainian
