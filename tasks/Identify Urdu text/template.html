Say whether Tweets are written in Urdu or not

  • This is a task for fluent speakers of Urdu.
  • Below are some short texts along with some decision options.
  • Please read each short text on the left.
  • On the right, mark each text for whether it is written for speakers of the Urdu language.
  • Also mark it as Urdu if:
    • it is a dialect of Urdu
    • it is in Urdu but written with English letters [abcdef...]
    • it is written for Urdu speakers but has some other languages mixed in.
  • If in doubt, mark it as Urdu if:
    • most of the text is in Urdu (excluding URLs, hash-tags, etc.).
    • Otherwise, mark as `Not Urdu'.
    • If still not sure, mark as 'Not sure' (rare cases).

Example:

  • "’عدلیہ کے کاندھے پر یہ کیا ناٹک ہے‘: عدلیہ کے کاندھے پر یہ کیا ناٹک ہے" is Urdu
  • "اعتصاب سراسری در نپال: اعتصاب سراسری در نپال، زندگی عادی را در این کشور مختل کرده است" is Not Urdu
