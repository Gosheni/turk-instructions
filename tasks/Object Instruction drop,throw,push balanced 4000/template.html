Thanks for participating in this HIT!

In this task, you will be asked to write instructions that describe physical interaction in a video.
Hopefully, a robot should be able to repeat the same action in video based on your written instructions.

Guidelines
  • You will be shown a video that involves some physical interaction with ONE object.
  • Provide precise instructions in 1-2 sentences that describe what is happening in the video.
  • NOTE: Use the word 'object' in your description to describe the object of interest (e.g. 'lift the object...'). See the example instruction.
  • But, you can name other object (e.g. drop the object on the 'desk') in your instruction other than the object of interest.
  • Please state all the actions happening in the video.
  • You do not have to state the outcome of the object (e.g. breaking/falling/rolling).
  • For now, try to come up with phrases that are accurate and different from the example instruction.
